# AndroidClaw - 智能视频广告识别与监测系统

AndroidClaw 是一款基于 Android 平台的端侧自动化视频广告监测工具。它利用 MediaProjection 屏幕采集技术与感知哈希（Perceptual Hashing）算法，实现对移动端 App 广告投放的实时精准侦听、匹配标记以及自动化片段导出。

## 🚀 核心特性

- **多目标并发侦听**：支持上传多个目标视频，系统并行运行多维指纹匹配算法，同时监控不同广告的投放情况。
- **高精度 dHash 匹配**：采用 64 位差异哈希（dHash）算法，支持在 8fps 的高频采集下进行秒级响应，有效对抗录屏帧偏色、轻微拉伸等干扰。
- **全自动任务闭环**：
    - **自动识别**：精准捕捉广告起止时间。
    - **自动裁剪**：任务完成后自动执行无损视频剪辑，保留广告触达的视频证据。
    - **自动终止**：队列任务全部达成后自动释放系统资源，确保低能耗运行。
- **工业级稳定性增强**：内置空闲帧（如全黑、全白转场）过滤逻辑，配合“持续时长+命中密度”双重验证，极大降低误报率。
- **交互式任务管理**：提供完整的 UI 控制台，支持手势删除任务、实时日志审计、结果视频直接回放。

## 🛠️ 技术架构

- **感知哈希工程化**：自研 `VideoFingerprintManager`，针对 Android 屏幕特征优化了灰度化与差分矩阵计算过程。
- **媒体处理管线**：
    - 使用 `MediaProjection` 实现非侵入式屏幕采集。
    - 结合 `VirtualDisplay` 与 `MediaRecorder` 实现多链路音视频同步录制。
    - 集成 `MediaCodec`/`MediaMuxer` 方案实现毫秒级精度的视频裁剪。
- **端侧资源管理**：基于 Kotlin 协程调度异步分析任务，充分利用移动端多核性能，保证采集不丢帧。

## 📋 快速开始

### 运行环境
- Android 10.0 (API 29) 或更高版本。
- 需开启“无障碍服务（Accessibility Service）”以支持系统调度。
- 需授权“屏幕录制”权限。

### 操作步骤
1. **添加目标**：点击主界面的“添加视频”按钮，上传需要监测的广告原片。
2. **启动服务**：确认无障碍服务开启后，点击“开始任务”。
3. **实时匹配**：在目标 App（如抖音、腾讯视频等）中进行日常操作。当目标广告出现时，系统会自动标记。
4. **查看结果**：任务完成后，在列表项中直接点击“查看结果”即可回放捕获到的广告证据片段。
